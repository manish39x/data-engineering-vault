## My Solution

```sql
-- Write your PostgreSQL query statement below
WITH previous_day_temp AS(
    SELECT
        id, temperature, recordDate,
        LAG(temperature) OVER(ORDER BY recordDate) AS yesterday_temp,
        LAG(recordDate) OVER(ORDER BY recordDate) AS yesterday_date
    FROM Weather
)
SELECT
    id
FROM previous_day_temp
WHERE temperature > yesterday_temp AND yesterday_temp IS NOT NULL
AND recordDate = yesterday_date + INTERVAL '1 day'
```
